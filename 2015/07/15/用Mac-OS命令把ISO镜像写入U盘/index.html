用Mac OS命令把ISO镜像写入U盘

Mac OS

1、用磁盘工具查看磁盘空间使用情况

~ ❯❯❯ df -h
Filesystem      Size  Used Avail Use% Mounted on
/dev/disk1      238G  154G   84G  65% /
/dev/disk2s2    15G  2.5M   15G   1% /Volumes/Centos
/dev/disk3      4.4G  4.4G     0 100% /Volumes/CentOS_6.6_Final

这里的/dev/disk2s2是我的U盘，容量16G

2、umount掉U盘

~ ❯❯❯ sudo diskutil umount /dev/disk2s2

3、将镜像写入U盘

~ ❯❯❯ sudo dd if=/Users/CBC/Desktop/CentOS-6.6-x86_64-bin-DVD1.iso of=/dev/disk2s2

4、用iostat命令查看磁盘写入状态

 ~ ❯❯❯ iostat -w 2
从Centos 6.x 升级Centos7

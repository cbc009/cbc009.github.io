TOP命令解释

top命令是Linux下最常用的性能分析工具，它能够实时显示系统中各个进程的资源占用状况。

top命令参数

-b   批处理模式（batch mode），可输出到管道、文件。默认情况下-b会一直输出，可以用-nN指定输出次数。
-n N限制输出次数。
-d N    刷新时间间隔。
-p PID  监控指定进程。
-H -p PID   监控指定进程和进行内线程信息。
